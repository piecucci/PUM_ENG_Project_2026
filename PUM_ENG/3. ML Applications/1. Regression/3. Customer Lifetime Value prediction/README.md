# Customer lifetime value prediction

The meaning of each column is as follows:

- customer_id: customer ID in the system
- age: age
- price_first_item_purchased: price of the first product he bought from you
- gender: gender
- join_ip: the IP address of the computer
- join_ip_country: country from which he logs in
- join_pages_visited: how many pages on average does he visit in one visit?
- join_campaign: did he come to you from a marketing campaign?
- join_GDP_cap: GDP per capita of the country where he or she lives
- revenue: total revenue. This is the variable you want to forecast
