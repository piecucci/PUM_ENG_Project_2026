# Missing values

To understand the principles of Data Science methods and techniques, it is useful to use so-called synthetic datasets.

They should be:

1. simple enough to replicate their operation on, for example, a piece of paper or a simple spreadsheet
2. represent as many different aspects of the data we are working on as possible.

The purpose of the following tasks is to practice different methods of completing missing data.



## Task 1. Complete the data in the synthetic - tshirts.csv file



The *synthetic - tshirts.csv* file is a synthetic dataset of various types with many missing data.

Test in Dataik the various methods for filling in missing data indicated in the Lecture.

1. log in to the Dataiku platform
Create a new project: *Missing_data*.
3) Import the *synthetic - tshirts.csv* file
4. test different methods of data completion.
