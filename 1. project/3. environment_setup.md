# Environment Setup

## Option 1: Google Colab (Recommended)

Google Colab is the primary supported environment.

### Steps for Colab

1. Read the instructions.
2. **Upload data files** to Google Drive:

   ```text
   My Drive/
   └── PUM/
       └── 2. data/
           ├── customers.csv
           └── transactions.csv
   ```

3. **Open the starter notebook** in Colab.
4. **Run the first cell** (reproducibility locks):

   ```python
   import sklearn, pandas as pd, numpy as np
   # Version assertions will pass on standard Colab
   ```

5. **Run the second cell** (data loading).

### Colab Data Path

The notebooks automatically detect Colab and set:

```python
DATA_DIR = Path("/content/drive/MyDrive/PUM/2. data")
```

### Mounting Google Drive

If prompted, mount your Drive:

```python
from google.colab import drive
drive.mount('/content/drive')
```

## Option 2: Local Setup

### Prerequisites

- Python 3.10+
- `uv` package manager

### Steps for Local

1. **Clone or download** the repository.
2. **Install dependencies**:

   ```bash
   cd PUM_ENG
   uv venv
   uv pip install pandas numpy scikit-learn matplotlib seaborn jupyter
   ```

3. **Verify library versions**:

   ```bash
   uv run python -c "
   import sklearn, pandas, numpy
   print(f'sklearn:  {sklearn.__version__}')
   print(f'pandas:   {pandas.__version__}')
   print(f'numpy:    {numpy.__version__}')
   "
   ```

4. **Start Jupyter**:

   ```bash
   uv run jupyter notebook "3. notebooks/"
   ```

## Troubleshooting

### "AssertionError: scikit-learn version..."

**Colab fix:**

```python
!pip install scikit-learn==1.5.2
# Restart runtime
```

**Local fix:**

```bash
uv pip install scikit-learn==1.5.2
```

### "AssertionError: numpy version..."

**Fix:**

```python
!pip install numpy==1.26.4
# Restart runtime
```

### "AssertionError: customers.csv MD5 mismatch"

Re-download `customers.csv`. The correct MD5 is `c016cd4bfc36ac8d0a99bb6a3d55fa44`.

### "FileNotFoundError: data directory not found"

**Colab:** Check path `My Drive/PUM/2. data/customers.csv`

**Local:** Run Jupyter from `3. notebooks/` so `../2. data/` resolves.

### "ModuleNotFoundError: No module named 'seaborn'"

**Colab:**

```python
!pip install seaborn
```

**Local:**

```bash
uv pip install seaborn
```

### My numbers don't match

1. Use `random_state=42` everywhere.
2. Follow the exact 12-step preprocessing order in MP2.
3. Run `3. notebooks/scripts/verify_checkpoint.py` to compare outputs.

### Google Colab session disconnects

Sessions time out after ~90 minutes of inactivity. Save your work frequently.
